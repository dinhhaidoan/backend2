# AcademicYear API (Khóa học)

Endpoints:
- GET /api/share/academic-years?page=1&limit=20&q=search
- GET /api/share/academic-years/:id
- POST /api/share/academic-years  (auth: staff/admin)
- PATCH /api/share/academic-years/:id (auth: staff/admin)
- DELETE /api/share/academic-years/:id (auth: staff/admin)

Request/response examples and guidance are available in the main docs.
